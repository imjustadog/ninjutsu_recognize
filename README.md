# ninjutsu_recognize
## 使用方式：  
1.本机安装opencv V3.4.2至/usr/local  
2.安装mlu220驱动  
3.安装cntoolkit  
4.运行make all进行编译  
5.运行./run.sh执行  
注：默认使用摄像头0，设备节点/dev/video0
